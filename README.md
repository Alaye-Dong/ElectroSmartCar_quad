## 项目简介

本项目为 **全国大学生智能汽车竞赛（电磁组）** 的四轮循迹小车，采用 **后轮驱动 + 前轮舵机转向** 方案，主控使用 `STC32G12K128`。通过工字电感采集赛道电磁信号，实现自主循迹。

> [!WARNING]
> **代码状态说明**：本仓库为大一首次参赛的工程版本，注释与结构尚显稚嫩。如需更成熟、工程化的代码体验，请移步改进版项目 [SmartCar_BST_Epoilt](https://github.com/Alaye-Dong/SmartCar_BST_Epoilt)。

> [!TIP]
> 此仓库为四轮车版本（后轮驱动 + 前轮舵机转向）。如需三轮车版本（后轮驱动 + 后轮差速转向），请前往 [SmartCar_BST_Epoilt](https://github.com/Alaye-Dong/SmartCar_BST_Epoilt)。
